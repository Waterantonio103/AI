# AI Experiments & Blender Addons

This repository is a collection of experiments with AI, focusing mainly on Blender add-ons generated or assisted by modern AI tools.

## Overview

This project explores how AI can be used to:
- Generate Blender add-ons
- Automate 3D workflows
- Assist with scripting and prototyping
- Experiment with AI-driven development pipelines

Most of the code in this repository is created using:
- Claude Code
- Google Gemini

These tools are used to accelerate development, test ideas, and push the limits of AI-assisted coding.

## Contents

- Blender add-ons (Python)
- AI-generated scripts
- Prototypes and tests
- Experimental workflows

Note: This is an experimental playground.

## Requirements

- Blender (3.x or newer recommended)
- Python (depending on addon)
- API keys (if using AI integrations like Gemini)

## Installation

1. Clone the repository:
   git clone https://github.com/Waterantonio103/AI.git

2. Open Blender
3. Go to Edit → Preferences → Add-ons → Install
4. Select the addon file from this repo
5. Enable it

## Usage

Each addon or script may have different usage instructions.

General workflow:
- Enable the addon in Blender
- Open the UI panel (usually in the sidebar)
- Run or test features
- Modify or experiment with the code
